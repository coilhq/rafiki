:::caution
If you are running Rafiki via Docker, you must update the variables in the `docker-compose.yml` files of the respective services.

If you are running Rafiki via Helm and Kubernetes, you must update the variables in the <LinkOut href='https://github.com/interledger/rafiki/blob/main/infrastructure/helm/rafiki/values.yaml'> values.yaml</LinkOut> file of the Helm chart template.

:::
